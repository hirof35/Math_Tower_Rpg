Numeric Tower Strategy
OpenSiv3Dで実装された、算術パズルとタワー攻略を融合させた戦略シミュレーションゲームです。
自身の「数値（Power）」をリソースとして管理し、敵の数値と演算子（加算・乗算）を考慮しながら、最適な攻略順序を選択する論理的思考が求められます。

🎮 ゲームシステム
基本ルール: プレイヤー（青い矩形）をドラッグして敵にぶつけることで戦闘を行います。

戦闘の論理:

加算（+）/ 減算（-）: 自身の数値が敵より大きい場合のみ勝利し、その数値を吸収します。

乗算（x）: 自身の数値を倍増させます。

クリア条件: フロア内のすべての敵ユニットを撃破するとステージクリアです。

ランキング: クリア時の到達レベルと経過時間をJSON形式でローカルに保存します。

✨ 技術的特徴
シーンマネージャーによる状態管理:

Siv3D::SceneManager を活用し、タイトル、ゲーム本編、クリア、ランキング、ゲームオーバーの遷移を構造的に管理。

データ永続化（JSON Serialization）:

ランキングデータを ranking.json に保存。ファイルが存在しない場合の例外処理や、配列構造の動的拡張を実装。

エフェクトシステム（IEffect）:

IEffect インターフェースを継承した「紙吹雪（Confetti）」エフェクト。物理演算（重力と速度）を用いたパーティクル描画を実現。

直感的なUI/UX:

SimpleGUI を用いたスピーディなUI実装と、ドラッグ＆ドロップによる直感的な操作性。

🛠 技術スタック
Language: C++20

Framework: OpenSiv3D (v0.6.x)

Architecture:

Scene-based architecture

Object-oriented enemy unit management

JSON-based persistence

📐 設計思想：数値の抽象化と戦略性
本プロジェクトは、複雑な戦闘を「数値の演算」という極めて抽象度の高いモデルに落とし込むことで、プレイヤーに純粋な論理パズルを提供することを意図しています。
これは、複雑な社会システムをクラスや演算子でモデル化する「思想工学」のプロトタイプ的アプローチを体現したものです。
